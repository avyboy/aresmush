    class AresMUSH::Bbs::PostTemplate

    Template code for showing a specific bbs post.

    Attributes

    replies[RW]

    List of all replies to this post, in order by date. You would typically use this in a loop, such as in the example below. Inside the loop, each reply would be referenced as 'r' and its counter index (0,1,2) as 'i'.

    <% replies.each_with_index do |r, i| %>
<%= reply_title(r) %>
<% end %>

def reply_title(reply)
  name = reply.author.nil? ? t('bbs.deleted_author') : reply.author.name
  date = OOCTime.local_long_timestr(@client, reply.created_at)
  t('bbs.reply_title', :name => name, :date => date)
end
